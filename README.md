# Salesforce O2C Sales Operations Project

This project is based on a sales operations, SLA monitoring and CRM workflow using Salesforce, Python, SOQL, and dashboard reporting.

In this project, I am working on 
- cleaning messy sales and customer data
- checking SLA and shipment issues
- finding duplicate and missing records
- running SOQL queries in Salesforce
- creating reports and dashboards

## Tools Used
- Python
- Pandas
- Salesforce Developer Org
- SOQL
- GitHub

## Workflow
1. Create messy sales data
2. Perform EDA and check data issues
3. Clean and fix the data using Python
4. Upload data into Salesforce
5. Run SOQL queries
6. Build reports and dashboards

## Key Project/Business Problems
- Missing and inconsistent CRM data
- Invalid shipment timelines
- SLA breaches
- Duplicate records
- Pricing inconsistencies
